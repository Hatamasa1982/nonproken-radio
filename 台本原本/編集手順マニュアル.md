# 📻 音声編集手順マニュアル（Filmora）

このドキュメントは「隣ラジオを振り返るラジオ」のFilmoraを使った音声編集の手順をまとめたものです。

## 🎬 編集の基本設定（ルール）
- **編集ツール**：Filmora（フィモーラ）
- **BGM音源**：そのまま使用
- **ZOOM音声**：そのまま使用
- **オーディオダッキング**：55%に設定
- **BGMテーマ**：Upbeat inside folk pop

---

## 🛠️ 編集の作業手順

### 1. 準備と音源の入れ替え
- [ ] Filmoraを開く。
- [ ] 前回編集したプロジェクトファイルを開く。
- [ ] 今回の新しい音源（ZOOM音声など）をインポートする。
- [ ] タイムライン上にある**前回の音源**を削除する。

### 2. トラックと音声の設定
- [ ] BGMのトラックを固定（ロック）する。
- [ ] インポートした今回の音源を作業画面（タイムライン）へ配置する。
- [ ] 今回の音源の**オーディオダッキングを55％**に設定する。

### 3. オープニングの編集
- [ ] オープニング部分を編集する（不要な音声をカットしながら調整）。
- [ ] トラック部分を固定する。
- [ ] オープニングの長さは**「7秒」**に合わせる。

### 4. タイトルと説明文の作成
- [ ] 今回のタイトルの設定
  > **【タイトルの記載例】**  
  > いつも隣にお仕事ラジオを振り返るラジオ#0000 番組紹介とパーソナリティ自己紹介
- [ ] 概要欄などに記載する説明文を作成・設定する。
